# プライバシーポリシー

**最終更新日:** 2026年3月22日

GitInflow（以下「本アプリ」）は hakaru が開発しています。本ポリシーでは、お客様のデータの取り扱いについて説明します。

## アクセスするデータ

### GitHub アカウントデータ
- 本アプリは OAuth 認証を通じて `repo` スコープで GitHub アカウントにアクセスします
- リポジトリ、Issue、ラベル、マイルストーン、コメントの読み書き権限が含まれます
- 認証トークンは iOS Keychain に安全に保存されます
- トークン交換はサーバーサイドプロキシ（Cloudflare Workers）を介して行われ、OAuth 認証情報が第三者サービスを経由することはありません

### デバイスに保存されるデータ
- **OAuth トークン:** iOS Keychain に保存（暗号化、ハードウェア保護）
- **リポジトリ設定:** 選択したリポジトリとデフォルトリポジトリを UserDefaults に保存
- **Issue キャッシュ:** 表示用の一時的な Issue データキャッシュ

## 収集しないデータ

- 個人データをサーバーに収集・保存・送信することは**ありません**
- アナリティクスやトラッキングサービスは**使用しません**
- 広告や広告トラッキング（IDFA）は**使用しません**
- アプリ内での表示以外の目的で GitHub データにアクセスすることは**ありません**

## サードパーティサービス

- **GitHub API:** Issue の取得・管理のために GitHub REST API と直接通信します。GitHub に保存されたデータには GitHub のプライバシーポリシーが適用されます。
- **Cloudflare Workers:** OAuth トークン交換にのみ使用。ユーザーデータの保存やログ記録は行いません。

## データの共有

第三者とデータを共有することはありません。

## データの削除

- [GitHub 設定 > アプリケーション](https://github.com/settings/applications)から、いつでも本アプリの GitHub アクセス権を取り消すことができます
- アプリをアンインストールすると、OAuth トークンを含むすべてのローカルデータが削除されます
- アプリの設定タブからログアウトすることで、保存されたトークンを削除できます

## お子様のプライバシー

本アプリは13歳未満のお子様を対象としていません。
