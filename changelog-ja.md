# 更新履歴

## v1.0.0 (2026-03-22)

初回リリース。

### 機能
- **カンバンステータスタブ** — Backlog / Ready / In Progress / Done
- **マルチリポジトリ対応** — 最大5個のリポジトリを選択、横断表示
- **クイックトリアージ** — スワイプでステータス変更、ラベル、マイルストーン、コメント
- **スマート共有シート** — 他のアプリからIssue作成
- **Pull Requestsタブ** — PR専用ビュー
- **Issue詳細** — コメント表示、ステータスバッジ、ステータス変更
- **Issue検索** — タブ内でタイトルによるフィルタ
- **ウィジェット** — ホーム画面にIssue数表示
- **ローカル通知** — 新しいIssueの通知
- **サーバーサイドOAuth** — Cloudflare Workers による安全なトークン交換
