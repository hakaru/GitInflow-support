# Changelog

## v1.0.0 (2026-03-22)

Initial release.

### Features
- **Kanban Status Tabs** — Backlog / Ready / In Progress / Done
- **Multi-Repository Support** — Select up to 5 repos, cross-repository view
- **Quick Triage** — Swipe actions for status change, labels, milestones, comments
- **Smart Share Sheet** — Create issues from any app
- **Pull Requests Tab** — Dedicated PR view
- **Issue Detail** — Comments, status badges, status change actions
- **Issue Search** — Filter issues by title within each tab
- **Widget** — Home screen widget with issue counts
- **Local Notifications** — New issue alerts
- **Server-side OAuth** — Secure token exchange via Cloudflare Workers
